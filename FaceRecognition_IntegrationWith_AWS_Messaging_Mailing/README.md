# Summer2021Task6

Task Description 📄

❄️ **Create a program that perform below mentioned task upon recognizing a particular face.** 

📌 When it recognize your face then :\
👉 It send mail to your mail id by writing this is face of your_name.\
👉 Second it send whatsapp message to your friend, it can be anything.

📌 When it recognize second face, it can be your friend or family members face.\
👉 Create EC2 instance in the AWS using CLI.\
👉 Create 5 GB EBS volume and attach it to the instance.

🚨 Note:\
👉 To Send Whatsapp Message you need to login through whatsapp web atleast once for complete flow of automation.\
👉 To Perfrom actions related to AWS , firstly we need to configure and login through our AWS CLI in task performing node.\
👉 For sending Mail using python as automation is not allowed by default in secured apps like gmail we need to allow less secure apps option in our gmail.
